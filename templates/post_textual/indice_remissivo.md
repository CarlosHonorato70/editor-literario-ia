## Informações de Uso

Este template é utilizado para gerar o índice remissivo com mais de 200 entradas.

### Variáveis Disponíveis:
- `{{index_entries}}` - Lista de termos e páginas

### Formato Esperado:
```
A

Análise estrutural, 15, 23, 45
Aprimoramento de conteúdo, 67-89
Autor, direitos do, 120

B

Bibliografia, 200-210
Blurb, criação de, 156

C

Capítulos, estruturação de, 34-56
Catalogação, 78
CIP, ficha, 12

...
```

### Notas:
- Entradas organizadas alfabeticamente
- Números de página separados por vírgulas
- Intervalos de páginas indicados com hífen
- O sistema gera automaticamente 200+ entradas
