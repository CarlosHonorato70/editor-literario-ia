## Informações de Uso

Este template é utilizado para gerar apêndices do manuscrito.

### Variáveis Disponíveis:
- `{{appendix_letter}}` - Letra do apêndice (A, B, C, etc.)
- `{{appendix_title}}` - Título do apêndice
- `{{appendix_content}}` - Conteúdo do apêndice

### Exemplo:
```
## Apêndice A - Questionário Utilizado na Pesquisa

[Conteúdo do questionário...]

## Apêndice B - Dados Estatísticos Complementares

[Tabelas e gráficos...]
```

### Notas:
- Apêndices contêm material elaborado pelo próprio autor
- Diferente de anexos (material de terceiros)
- Numerados com letras maiúsculas: A, B, C, etc.
