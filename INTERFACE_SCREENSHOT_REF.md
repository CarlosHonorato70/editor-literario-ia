## Responsiveness

The interface adapts to different screen sizes:
- **Desktop**: Side-by-side columns for options and preview
- **Tablet**: Stacked layout with full-width sections
- **Mobile**: Vertical scrolling, full-width components

## User Flow

1. User arrives at Tab 2
2. Sees clear explanation of what FastFormat does
3. Chooses a preset (default: PT-BR)
4. Clicks "Preview Formatting" button
5. Sees before/after comparison
6. Decides to apply or discard
7. If applied, returns to Tab 1 with formatted text
8. Can repeat process as many times as needed

## Accessibility

- Clear visual hierarchy
- Large, easy-to-click buttons
- High contrast text
- Descriptive labels
- Keyboard navigation support
- Screen reader friendly

## Performance

- Preview generation: < 1 second for typical texts
- Smooth animations on state changes
- No page reload needed
- Efficient text comparison display
